# AI Workflow Pivot Database

This root directory houses the massive knowledge catalog for D-Tul's pivot to "AI Workflow Specializations". As a >3 day project, this environment will act as the "Brain" of our service offerings.

## Structure
- `/case_studies/`: Markdown files of real-world enterprise automations and system migrations.
- `/automation_logics/`: System architecture flowcharts, mermaid diagrams, and logic maps for specific automations.
- `/jsons/`: Data structures, schemas, and configurations for scalable workflows.
- `/agents/`: Specialized agent definitions required for complex automated tasks.

## Objectives
Synthesize 100s of case studies to strictly define:
1. **Who we sell to** (ICP)
2. **What we deliver** (Tech Stack)
3. **How much we charge** (Value-based pricing)
4. **How long it takes** (Delivery pipeline)
